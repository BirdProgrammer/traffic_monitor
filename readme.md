## Traffic  Monitor

### 运行参数

| 参数 | 含义                                                     |
| ---- | -------------------------------------------------------- |
| p    | 欲使用的端口，掩码格式                                   |
| t    | 显示时间间隔，单位为秒                                   |
| file | 配置文件名称，配置文件包含流数目以及各个流所包含的目的IP |

例如：

```
./build/trafficMonitor -c f -- -p 3 --file=flow_info.conf -t 3
```

### 运行结果

每隔t秒，输出这段时间内所有流的平均速率。
